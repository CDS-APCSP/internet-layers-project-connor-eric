Section 1

The presentation layer is the second to last layer coming before the application layer in the OSI model.

Section 2

The presentation layer is in charge of get data ready to be used by the aplication layer. This layer, translates, encrypts, and compresses data. In other words, it "presents" data to the aplication layer.

If two devices that are communicating are using two different communication methods, the presentation layer will translate the the data into a syntax that the aplication layer will understand and will show to the recieving device.
